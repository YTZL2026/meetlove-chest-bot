# 方案A：手机党（Auto.js）

## A1. 装 Auto.js（3分钟）

由于 Auto.js 被下架了，推荐用 **AutoX.js v7**：

> ⚠️ 原 `kkevsekk1/AutoX` 已停止维护，社区已迁移至 `aiselp/AutoX` (v7)

1. 手机浏览器打开：`https://github.com/aiselp/AutoX/releases`
2. 下载最新的 `.apk` 文件（选 `arm64-v8a` 版本，旧手机选 `armeabi-v7a`）
3. 安装，手机会提示「未知来源」→ 点「允许」
4. 装好后打开 App

## A2. 开权限（1分钟）

打开 AutoX.js 后会提示开权限，跟着点：

1. **无障碍服务** → 点「去开启」→ 找到 AutoX.js → 打开开关
2. **悬浮窗权限** → 点「去开启」→ 找到 AutoX.js → 允许

> 不同手机叫法可能不一样：悬浮窗/在其他应用上层显示/画中画，都是一个东西，找到允许就行。

## A3. 导入脚本（2分钟）

1. 把 `chest_farm_main.js` 文件传到手机上（微信发给自己 / QQ 传文件 / 数据线）
2. 打开 AutoX.js
3. 右下角 **"+"** → **导入** → 找到刚才传的文件 → 点它

## A4. 改几个数字（2分钟）⚠️ 最重要的一步

在 AutoX.js 里点开脚本，找到最前面的配置区，改这 4 个东西：

### ① 改分辨率（必改！不改脚本点不到正确位置）

```
你的手机分辨率怎么查？
→ 百度搜「[你的手机型号] 屏幕分辨率」
→ 或者去手机「设置 → 关于手机」里找
```

找到脚本里这行，改成你的：

```javascript
screen: {
    width: 1080,    // ← 改成你的屏幕宽度
    height: 2400,   // ← 改成你的屏幕高度
},
```

### ② 改游戏包名（必改！）

找到这行：

```javascript
let expectedPkg = "com.meetlove.android"; // ← 改成遇见的实际包名
```

**怎么查包名？**
- 打开 AutoX.js → 左侧菜单 → **「应用」** → 找到「遇见」→ 点进去 → 复制「包名」
- 如果找不到，这行可以先不改，脚本也能跑（只是不会检测游戏是否在后台）

### ③ 改切图坐标（关键！决定能不能切图成功）

**先在你的游戏里试一下**：站在家门口（坐标 2152, 3161 附近），手指在屏幕上**往菜地方向滑**，看看跨过地图边界需要手指滑多远。

脚本里改这里（坐标按比例换算）：

```javascript
waypoints: [
    {
        name: "家门↔菜地",
        x1: 540, y1: 1600,  // ← 手指起点（屏幕中间偏下）
        x2: 540, y2: 800,   // ← 手指终点（屏幕中间偏上）
        // 意思：手指从下往上滑，角色往北走
    },
```

> **懒人办法**：如果你的手机分辨率和 1080x2400 差不多（比如 1080x2340），直接用默认值试一次，能切图就说明没问题，不用改。

### ④ 改击杀速度

```javascript
killTime: 1500,   // 你能秒杀宝箱怪 → 改成 500
                   // 你需要打 2-3 下  → 改成 2000
                   // 你需要打 5 下以上 → 改成 4000
```

> 改完记得点右上角 **保存**！

## A5. 开始挂机（1分钟）

1. 打开「遇见」游戏
2. 控制角色站在切图边界（比如家门口坐标 2152, 3161）
3. **开启游戏内置挂机**（设置里关掉采集，打开自动战斗）
4. 画质调到**最低**
5. 切回 AutoX.js → 点脚本 → 点 **运行** ▶️
6. 会弹出一个悬浮小窗，显示日志

## A6. 怎么停止？

- **按手机音量上键**（默认）
- 或者点悬浮窗上的 **X**
- 或者在 AutoX.js 里点 **停止** ⏹

# 方案B：电脑模拟器党（ADB）

## B1. 装 Python（5分钟）

1. 打开 `https://www.python.org/downloads/`
2. 点黄色大按钮 **Download Python**
3. 下载完后双击安装
4. **⚠️ 重要**：安装界面最下面勾选 **「Add Python to PATH」**，再点 Install
5. 装完点 Close

## B2. 确认 ADB 能用（2分钟）

1. 打开模拟器（雷电/MuMu/蓝叠）
2. 模拟器设置里找到 **ADB 调试** → 打开
3. 电脑上按 `Win + R` → 输入 `cmd` → 回车
4. 在弹出的黑窗口里输入：

```
adb devices
```

5. 如果看到类似这样的输出：

```
List of devices attached
127.0.0.1:5555    device
```

说明成功了！如果看不到，试试：

```
雷电模拟器：adb connect 127.0.0.1:5555
MuMu模拟器：adb connect 127.0.0.1:7555
蓝叠模拟器：adb connect 127.0.0.1:21503
```

## B3. 改脚本配置（3分钟）

用「记事本」打开 `chest_farm_adb.py`，改最前面的配置：

### ① 模拟器分辨率

```python
SCREEN_W = 1080    # ← 改
SCREEN_H = 2400   # ← 改
```

模拟器设置里能看到分辨率，设置成 1080x2400 最省事。

### ② 切图坐标

和方案A一样，需要根据分辨率调整。1080x2400 可以直接用默认值。

### ③ 击杀速度

```python
KILL_TIME = 1500   # 秒杀→500，打几下→2000，打很久→4000
```

## B4. 开始挂机（1分钟）

1. 模拟器里打开「遇见」游戏
2. 站在切图边界，开启游戏内置挂机
3. 画质调最低
4. 电脑上打开 cmd（Win+R → cmd → 回车）
5. 进入脚本目录：

```
cd "C:\Users\86132\AppData\Local\Programs\CC Switch\meetlove-chest-bot"
```

6. 运行：

```
python chest_farm_adb.py
```

7. 看到 `🚀 宝箱怪挂机脚本启动` 就说明开始跑了

## B5. 怎么停止？

在 cmd 窗口按 **Ctrl + C**。

# 常见问题（遇到问题先看这里）

## ❓ 游戏里角色不动/不切图？

检查这几样：
- 游戏画质是不是调到**最低**了？（高画质下地面物体遮挡可能导致挂机异常）
- 游戏内置挂机打开了吗？（设置 → 挂机 → 开）
- 切图坐标对了吗？（试着用手指在脚本设置的那个坐标滑一下，能不能跨地图？）

## ❓ 切图成功了，但不杀宝箱怪？

- 你的战力能秒杀宝箱怪吗？不能的话把 `killTime` 调大
- 检查游戏挂机设置：「自动战斗」打开，「采集」关掉
- 检查武器射程：弓/法杖最好，近战武器可能打不到

## ❓ 跑着跑着人物跑去奇怪的地方了？

- 这是游戏内置挂机寻怪导致的，正常现象
- 只要它还能源源不断地切回边界就行
- 如果跑太远回不来 → 脚本有「卡死检测」会自动复位
- 不影响刷宝箱怪效率

## ❓ 脚本提示「无障碍服务未开启」？

去手机「设置 → 辅助功能 → 无障碍」→ 找到 AutoX.js → 打开。
如果已经开了，试试关掉重开。

## ❓ 一晚上能刷多少只？

> ⚠️ 效率瓶颈不在脚本，而在**游戏机制**：每次切图虽然会刷新宝箱怪，但它不一定刷在角色身边，挂机AI也可能先追普通怪。以下为社区实测数据参考：

| 你的配置 | 完整周期 | 每小时 | 8小时（一晚） |
|----------|----------|--------|--------------|
| 秒杀+好设备 | ~15秒 | ~100-150只 | ~800-1200只 |
| 正常配置 | ~22秒 | ~80-100只 | ~600-800只 |
| 慢设备/低战力 | ~30秒 | ~50-70只 | ~400-500只 |

> 📌 社区实测：纯手动约 60只/h，脚本挂机平均约 50只/h。以上为切图法的优化预估，实际受宝箱怪刷新位置 RNG 和挂机 AI 锁敌优先级影响，浮动 ±30%。

## ❓ 会不会被封号？

这个脚本**只是模拟滑屏**，没有任何内存修改、加速、破解。相当于一个机械手帮你不停划屏幕。目前社区没有人因为这个方式被封。

但**不要 24 小时不间断挂**，建议：
- 晚上睡觉挂 6~8 小时，白天正常玩
- 偶尔手动玩一下，不要永远在挂机
