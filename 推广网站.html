📘 使用指南

1 安装 AutoX.js（1分钟）

手机浏览器打开：https://github.com/kkevsekk1/AutoX/releases，下载最新 .apk 安装

2 开启权限（30秒）

按弹窗提示依次开启：无障碍服务 + 悬浮窗权限

3 导入脚本（30秒）

把 chest_farm_main.js 传手机 → AutoX.js 右下角【＋】→ 导入 → 选中文件

4 修改配置（2分钟）

配置项说明
屏幕分辨率1080x2400不用改，其他需查手机型号填写
击杀速度秒杀→500  |  打2-3下→2000  |  打5下以上→4000
切图坐标1080分辨率直接用默认值
💡 改完一定要点右上角【保存】！

5 开始挂机（30秒）

① 打开遇见 → 站在家门口（坐标 2152, 3161）

② 开游戏内置挂机：只开战斗，关采集

③ 画质调到最低

④ 回到 AutoX.js → 点运行 ▶

⏹ 停止挂机

按手机音量上键即可一键停止

1 安装 Python（2分钟）

python.org 下载安装，务必勾选 Add Python to PATH

2 开启ADB（1分钟）

模拟器设置 → 开启ADB调试 → Win+R 输入 cmd → 输入 adb devices

模拟器连接命令
雷电adb connect 127.0.0.1:5555
MuMuadb connect 127.0.0.1:7555
蓝叠adb connect 127.0.0.1:21503

3 修改配置（1分钟）

记事本打开 chest_farm_adb.py，改：分辨率 + 击杀速度 + 切图坐标

4 运行 & 停止

cd 你的脚本目录

python chest_farm_adb.py

看到 🚀 宝箱怪挂机脚本启动 即开始运行

停止：按 Ctrl + C

❓ 常见问题

角色不动 / 不切图怎么办？▼

① 游戏画质是否调到最低？

② 游戏内置挂机是否已开启？

③ 切图坐标是否正确？

切图成功但不杀宝箱怪？▼

① 战力能否秒杀？不能则把 killTime 调大

② 确认挂机设置：自动战斗开，采集关

③ 弓/法杖最好，近战可能打不到

会不会被封号？▼

脚本仅模拟滑屏，不修改内存/加速/破解。社区目前无人因此被封。建议每晚挂6-8小时，白天正常玩。

人物跑偏了？▼

游戏内置挂机寻怪导致，正常现象。脚本已内置卡死检测，跑太远会自动复位。

「无障碍服务未开启」？▼

设置 → 辅助功能 → 无障碍 → 找到 AutoX.js → 打开。已开则尝试关掉重开。

支持模拟器多开吗？▼

ADB版支持：adb devices 查看所有实例serial，修改对应 ADB_DEVICE，每个实例跑一个进程。
