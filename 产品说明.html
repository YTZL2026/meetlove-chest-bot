📘 方案 A：手机党（AutoX.js v7）5 分钟搞定

A1 装工具（1 分钟）

1. 手机浏览器打开：https://github.com/aiselp/AutoX/releases

2. 下载最新 .apk（主流手机选 arm64-v8a，旧手机选 armeabi-v7a）

3. 安装时允许「未知来源」→ 打开 AutoX.js

⚠️ 原 kkevsekk1/AutoX 已停止维护（旧链接已失效），现使用社区维护的 v7 分支

A2 开权限（30 秒）

按弹窗提示开启：无障碍服务 + 悬浮窗 → 找到 AutoX.js 打开开关即可

A3 导脚本（30 秒）

1. 把 chest_farm_main.js 微信发给自己并保存到手机

2. AutoX.js 点右下角【+】→ 导入 → 选中文件

A4 改配置（2 分钟，只改 4 个数字）

📐 屏幕分辨率 1080 开头不用改，其他查手机型号填写
⚔️ 击杀速度 秒杀填 500｜打 2-3 下填 2000｜打 5 下以上填 4000
🗺️ 切图坐标 1080 分辨率直接跳过，其他按比例换算
🐴 坐骑按钮坐标 新手必改！见下方「坐骑设置」说明
✅ 改完一定要点右上角【保存】！

🐴 A4.5 坐骑设置（新手关键！）

宝箱怪受惊后 8 秒 钻地消失。无称号/无速度装的新手跑步追不上，必须上坐骑。

1. 在 AutoX.js 中开启「指针位置」（悬浮窗 → 指针位置）

2. 切到游戏，手指点一下坐骑按钮，看屏幕上显示的坐标

3. 把坐标填到脚本里的 mount.btnX 和 mount.btnY

4. 常见位置参考（1080×2400）：

机型/布局大致坐标
右下角小图标1000, 2100
右侧中间圆形按钮1030, 1200
底部技能栏旁边950, 1950
⚠️ 没有坐骑的新手：脚本里把 mount.enabled 改成 false，否则会空点屏幕

A5 开始挂机（30 秒）

1. 打开遇见 → 站在切图边界（如家门口 2152, 3161 或心灵草原 1953, 1927）

2. 开游戏内置挂机：✅ 自动战斗，❌ 关采集

3. 画质调到最低（草变矮更容易看到宝箱怪）

4. 回到 AutoX.js → 点脚本 → 点【运行】▶️

A6 停止挂机

按 音量上键 即可一键停止 🛑

💻 方案 B：电脑模拟器（ADB）5 分钟搞定

B1 装 Python（2 分钟）

1. 官网下载安装：python.org

2. 必勾选：Add Python to PATH

B2 开 ADB（1 分钟）

1. 模拟器设置里开启 ADB 调试

2. Win+R 输入 cmd 回车

3. 输入命令：adb devices

4. 连不上就用对应端口命令：

模拟器连接命令
雷电adb connect 127.0.0.1:5555
MuMuadb connect 127.0.0.1:7555
蓝叠adb connect 127.0.0.1:21503

B3 改配置（1 分钟）

记事本打开脚本，改：分辨率 + 击杀速度 + 坐骑按钮坐标（重要！见方案A的坐骑设置说明）。模拟器建议直接设置分辨率 1080×2400 最省事。

B4 开始挂机

cmd 里依次输入两行命令回车：

cd 你的脚本所在文件夹路径

python chest_farm_adb.py

B5 停止挂机

按 Ctrl + C

🐴 v2.2 坐骑版：脚本在切图后自动上坐骑，新手也能追上宝箱怪。有坐骑的玩家实际效率可提升 30-50%（原先追不上→现在能追上）。

💡 游戏内置挂机刷新计时器为 16-25 秒，切图可绕过该限制强制刷新，但地图加载 + 索敌 + 击杀仍需要时间。

🛠 遇到问题？傻瓜式解决

角色不动/不切图 游戏画质必须调到【最低】；检查切图坐标是否匹配分辨率
切图不打怪 把击杀速度数字调大；确认武器为弓/法杖（远程）；确认游戏挂机开了自动战斗
无障碍未开启 设置 → 辅助功能 → 无障碍 → 找到 AutoX.js 重启开关
追不上宝箱怪 新手常见！开启脚本坐骑功能🐴，确认坐骑按钮坐标正确。没坐骑则关掉该功能
脚本报错"y1 重复" 已修复（v2.2），下载最新版脚本即可
跑着跑着去奇怪地方 游戏挂机寻怪导致，正常现象。脚本有卡死检测会自动复位
⚠️ 安全提示：每天挂 6-8 小时即可，不要 24 小时不间断。本脚本仅模拟滑屏操作，无内存修改/加速，社区目前无人因此被封。但保险起见，偶尔手动玩一下，不要永远在挂机。
